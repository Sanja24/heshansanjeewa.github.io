01. Problem Statement

In pharmaceutical environments, manual dispensing of medications is prone to human error and inefficiency. This project, initially conceptualized as "Pillpilot," was developed to automate the sorting and dispensing of tablets based on digital prescriptions. The goal was to create a high-precision, reliable system that bridges digital medical records with physical hardware execution.

02. System Architecture

The system operates through a multi-layered architecture, integrating high-level software logic with low-level hardware control.

Software Layer (Python)

Developed using PyQt5, the GUI parses prescription text files in real-time. It maps medication names to physical bin coordinates using a dynamic configuration database, ensuring that any bin layout can be accommodated without hard-coding locations.

Communication Protocol

The system utilizes G-code for machine instructions. By translating prescription data into standardized CNC commands, the system achieves universal compatibility and high-precision movement control over the Serial interface.

Hardware & Control (Arduino)

An Arduino-based firmware interprets the G-code to drive NEMA stepper motors. This ensures sub-millimeter accuracy for the dispensing head as it navigates the bin array to retrieve and dispense specific dosages.

03. Key Engineering Features

  • Real-time Prescription Parsing: Automated reading of `.txt` prescriptions to eliminate manual entry errors.
  • Scalable Bin Configuration: The software supports custom bin mapping, allowing the hardware to be scaled for larger pharmacies.
  • Safety Interlocks: Integrated logic to verify tablet selection before dispensing, ensuring pharmaceutical compliance.

04. Technical Outcomes

TAB SORT 1.9 demonstrates a successful integration of PyQt5 GUI development, G-code automation, and embedded systems. The project showcases the transition from a software-based medicine manager to a fully realized mechatronic solution capable of physical task execution.
