Plant Nutrient Assessment System (NUTRILEAF)

A portable, low-cost system designed to detect nutrient deficiencies in corn leaves using image processing and deep learning techniques.

Technologies

  • Python, OpenCV, TensorFlow
  • Portable microscope camera
  • Machine learning classification

Outcome

Enabled non-destructive, real-time nutrient analysis suitable for field use.
