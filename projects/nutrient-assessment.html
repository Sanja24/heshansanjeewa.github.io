Plant Nutrient Assessment System (NUTRILEAF)

Mechatronics | Computer Vision | Deep Learning

A portable, low-cost system designed to detect nutrient deficiencies in corn leaves using image processing and deep learning techniques. This project focuses on bridging the gap between lab-grade analysis and real-world farming environments.

×

Technologies Involved

  • Software: Python, OpenCV, TensorFlow/Keras for classification.
  • Hardware: Portable microscopic camera, customized lighting mount.
  • Method: Non-destructive image acquisition and real-time inference.

Key Outcome

The system successfully enabled real-time, non-destructive nutrient analysis suitable for field use, providing farmers with immediate data to optimize fertilizer application.
